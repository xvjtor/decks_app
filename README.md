# Decks App

Aplicação mobile desenvolvida em Flutter para criação de decks de estudos utilizando flashcards interativos.

---

## Preview

Sistema de estudos baseado em decks e cartões de perguntas e respostas.

O aplicativo permite criar decks personalizados, adicionar cartões e iniciar quizzes interativos para reforço de aprendizagem.

---

## Funcionalidades

- Criação de decks de estudo
- Adição de cartões personalizados
- Amazenamento local de dados
- Sistema de quiz interativo
- Contagem de acertos
- Resultado final da sessão de estudo
- Atualização reativa da interface

---

## Tecnologias Utilizadas

### Flutter
Framework utilizado para construção da interface mobile.

### Dart
Linguagem principal da aplicação.

### Hive
Banco de dados local utilizado para persistência dos decks e cartões.

### MobX
Gerenciamento de estado reativo.

### GetIt
Injeção de dependências e organização da arquitetura.

---

## Arquitetura

O projeto foi estruturado utilizando separação por camadas:

- Pages
- Widgets
- Stores
- Repositories
- Models

A arquitetura foi organizada para facilitar manutenção, escalabilidade e reutilização de código.
